# FoodExpress - App de Delivery

Aplicativo de delivery para bares e restaurantes, feito em **React Native (Expo)**.

## Funcionalidades

- Listagem de produtos
- Adicionar itens ao carrinho
- Remover itens do carrinho
- Finalizar pedido
- Integração com backend fictício

## Tecnologias

- React Native / Expo
- TypeScript
- React Navigation
- Hooks
- Axios
