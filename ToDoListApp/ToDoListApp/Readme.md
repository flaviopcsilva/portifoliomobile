# Todolist App - Gerenciador de Tarefas

Aplicativo mobile em **React Native (Expo)** para criar, editar, concluir e remover tarefas.

## Funcionalidades

- Criar tarefas
- Editar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Lista persistente localmente (AsyncStorage)

## Tecnologias

- React Native / Expo
- TypeScript
- AsyncStorage
- React Navigation
