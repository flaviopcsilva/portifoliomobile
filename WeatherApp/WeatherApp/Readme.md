# ClimaApp - Previsão do Tempo

Aplicativo mobile que consome a **API do OpenWeather** para exibir previsão do tempo.

## Funcionalidades

- Buscar cidade pelo nome
- Exibir temperatura, umidade e clima
- Forecast de 5 dias (3h)
- Mensagens de erro para cidade não encontrada ou API inválida

## Tecnologias

- React Native / Expo
- TypeScript
- Axios
- OpenWeather API

## Como executar

1. Instale as dependências:

npm install
